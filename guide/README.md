# 快速开始

## 什么是Docsify？

Docsify是一个动态生成文档网站的工具。不同于GitBook、Hexo等工具需要预编译，Docsify是一个轻量级的文档系统，可以快速帮助你生成文档网站。

### 特性

- 🚀 无需构建，写完即部署
- 📝 支持Markdown语法
- 🎨 多种主题可选
- 🔍 内置全文搜索
- 📱 响应式设计
- 🌍 多语言支持
- ⚡ 插件系统丰富

## 为什么选择Docsify？

### 简单易用

只需要创建一个`index.html`文件和编写Markdown文档即可，无需复杂的配置和构建流程。

### 轻量快速

运行时解析Markdown，不需要预先编译，文档更新后刷新即可看到效果。

### 功能强大

虽然简单，但功能一点也不弱，支持搜索、主题、插件等各种功能。

## 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- Node.js（如需本地服务器）
