## 🚀 快速开始

### 新手必备3件套

1. **ChatGPT / Kimi**
   - 日常对话、问答
   - 学习助手
   - 工作辅助

2. **Cursor / GitHub Copilot**
   - 编程辅助
   - 代码生成
   - Bug修复

3. **Midjourney / 文心一格**
   - 图片生成
   - 设计灵感
   - 视觉创作

### 5分钟上手

#### 1. ChatGPT快速体验

```
1. 访问 chat.openai.com（或使用Kimi）
2. 注册账号
3. 开始对话
4. 尝试各种任务：写作、翻译、编程等
```

#### 2. AI绘画初体验

```
1. 使用文心一格（无需魔法上网）
2. 输入提示词：一只可爱的橙猫，卡通风格
3. 生成图片
4. 调整参数重新生成
```

#### 3. AI编程助手

```
1. 安装Cursor或VS Code + Copilot
2. 写注释描述需求
3. AI自动生成代码
4. 修改和优化
```

## 💡 使用技巧

### 通用技巧

1. **清晰表达**
   - 具体描述需求
   - 提供足够上下文
   - 分步骤引导

2. **迭代优化**
   - 不满意就重新生成
   - 逐步调整提示
   - 结合不同工具

3. **验证结果**
   - AI可能出错
   - 关键信息需验证
   - 理解而非照搬

### 提示词模板

**对话模板**：
```
角色: 你是一个[专业领域]专家
任务: [具体任务描述]
要求: [输出格式、风格等]
示例: [可选，提供示例]
```

**编程模板**：
```
# 需求描述
功能: [具体功能]
输入: [输入参数]
输出: [期望输出]
限制: [性能、兼容性等]
```

**绘画模板**：
```
主体 + 风格 + 细节 + 参数

例: 一只橙猫, 赛博朋克风格, 未来城市背景, 霓虹灯效果, 高清, 8K
```
