## 🚀 快速入门

### 注册与登录

**方式一：官网注册**（需要魔法上网）

1. 访问 https://chat.openai.com
2. 点击"Sign up"注册
3. 输入邮箱（建议Gmail）
4. 验证邮箱
5. 设置密码
6. 完成注册

**方式二：使用国内镜像站**

- 搜索"ChatGPT镜像站"
- 选择可信赖的平台
- 注意数据安全

**方式三：API调用**

```python
import openai

openai.api_key = "your-api-key"
response = openai.ChatCompletion.create(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

## 📊 版本对比

### GPT-3.5 vs GPT-4

| 特性 | GPT-3.5 Turbo | GPT-4 |
|------|---------------|--------|
| **价格** | 免费 | $20/月 |
| **速度** | 快 ⚡⚡⚡ | 较慢 ⚡⚡ |
| **智能** | 优秀 ⭐⭐⭐⭐ | 卓越 ⭐⭐⭐⭐⭐ |
| **上下文** | 16K tokens | 128K tokens |
| **推理能力** | 好 | 非常强 |
| **代码能力** | 好 | 优秀 |
| **多模态** | ❌ | ✅ 支持图片 |
| **联网** | ❌ | ✅ |
| **文件上传** | ❌ | ✅ |

### 如何选择

**选择GPT-3.5**：
- ✅ 日常对话
- ✅ 简单问答
- ✅ 文本生成

**选择GPT-4**：

## 💡 核心功能

### 1. 对话聊天

**基础对话**：
```
用户: 介绍一下量子计算
AI: 量子计算是利用量子力学原理...
```

**多轮对话**：
```
用户: 什么是机器学习？
AI: 机器学习是...

用户: 它和深度学习有什么区别？
AI: 深度学习是机器学习的子集...（能记住上下文）
```

### 2. 文本创作

**文章写作**：
```
提示：写一篇800字的文章，主题是"人工智能对教育的影响"，
包含引言、主体、结论三部分。
```

**邮件起草**：
```
提示：帮我写一封求职邮件，应聘前端开发职位，
突出我的React和TypeScript经验。
```

**创意写作**：
```
提示：写一个科幻短故事，主角是AI机器人，
背景设定在2050年的火星殖民地。
```

### 3. 代码编程

**代码生成**：
```python
提示：用Python写一个函数，计算斐波那契数列的第n项

回答：
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

**代码解释**：
```
提示：解释这段代码的作用
[粘贴代码]

回答：这段代码实现了...
```

**Bug修复**：
```
提示：这段代码有什么问题？
[粘贴出错代码]

回答：问题在于...应该改成...
```

### 4. 数据分析

**GPT-4可用**（需要Plus）：
```
提示：分析这个CSV文件的数据，给出销售趋势
[上传文件]

回答：
1. 总体销售呈上升趋势
2. Q4季度销售最高
3. [生成图表]
```

### 5. 图片理解

**GPT-4V功能**：
```
提示：这张图片里有什么？
[上传图片]

回答：图片中显示了...
```

**多图对比**：
```
提示：对比这两张设计稿的差异
[上传两张图]

回答：主要差异有...
```

### 6. 文档处理

**PDF分析**：
```
提示：总结这份研究报告的要点
[上传PDF]

回答：
1. 研究目标
2. 方法论
3. 主要发现
```

**长文档问答**：
```
提示：根据这份合同，违约条款是什么？
[上传合同]

回答：根据文档第X条...
```

## 🎯 使用技巧

### 基础技巧

**1. 清晰表达需求**

❌ 不好的提示：
```
写个文章
```

✅ 好的提示：
```
写一篇1000字的科普文章，介绍区块链技术，
面向没有技术背景的读者，用通俗易懂的语言。
```

**2. 提供上下文**

❌ 不好：
```
优化这段代码
[代码]
```

✅ 好：
```
这是一个Node.js后端API接口，
性能较慢，需要优化响应时间。
目标：将响应时间从2秒降到500ms以内。
[代码]
```

**3. 分步骤引导**

❌ 不好：
```
帮我做一个完整的网站
```

✅ 好：
```
我要做一个个人博客网站。
第一步：请帮我设计网站结构和页面布局
```

**4. 要求特定格式**

```
提示：列出学习Python的步骤，
用表格形式，包含：步骤、内容、预计时间
```

**5. 使用示例**

```
提示：将以下内容改写成新闻稿格式

示例：
原文：我们公司发布了新产品
新闻稿：[公司名]今日宣布推出革新性产品...

现在改写：
[你的内容]
```

### 高级技巧

**1. 角色扮演**

```
提示：你是一位资深的Python开发工程师，
有15年经验，擅长代码架构设计。
请从架构角度评价我的代码。

[代码]
```

**2. 思维链（Chain of Thought）**

```
提示：请一步步分析这个问题，
展示你的思考过程：

如何设计一个高并发的秒杀系统？
```

**3. 自我反思**

```
提示：
1. 给出你的答案
2. 分析答案可能存在的问题
3. 给出改进版本
```

**4. 对比分析**

```
提示：对比React和Vue框架，
用表格列出：学习曲线、性能、生态、适用场景
```

**5. 迭代优化**

```
第1轮：写一个产品介绍
第2轮：让它更生动有趣
第3轮：加入用户痛点
第4轮：添加行动号召
```

## 📝 提示词工程

### 通用模板

**结构化提示**：
```
角色：你是[专业角色]
背景：[提供上下文]
任务：[明确任务]
要求：[具体要求]
格式：[输出格式]
示例：[可选]
```

**实例**：
```
角色：你是一位经验丰富的产品经理
背景：我们正在开发一款面向大学生的时间管理App
任务：撰写产品需求文档（PRD）
要求：
  - 包含用户画像
  - 核心功能列表
  - 优先级排序
  - 大约1500字
格式：Markdown格式，使用标题和列表
```

### 分类模板

**文章写作**：
```
写一篇关于[主题]的文章
目标读者：[读者群体]
字数：[字数]
风格：[正式/轻松/专业等]
结构：
  - 引言
  - 主体（3-5个要点）
  - 结论
要求：[其他要求]
```

**代码开发**：
```
用[编程语言]实现[功能]
输入：[输入参数]
输出：[期望输出]
要求：
  - 代码注释完整
  - 错误处理
  - 性能优化
  - 单元测试
```

**数据分析**：
```
分析[数据类型]数据
数据源：[描述或上传]
分析目标：[要了解什么]
输出要求：
  - 关键指标
  - 趋势分析
  - 可视化建议
  - 行动建议
```

### Few-Shot学习

**示例**：
```
将产品描述改写成营销文案。

例子1：
产品：高端蓝牙耳机，降噪功能
文案：沉浸纯净音质，隔绝喧嚣世界

例子2：
产品：智能手环，心率监测
文案：全天候守护健康，让运动更科学

现在改写：
产品：机械键盘，青轴
文案：
```

## ⚠️ 使用限制与注意事项

### 限制

**GPT-3.5**：
- 无法联网
- 知识截止2021年9月
- 上下文长度16K

**GPT-4**：
- 每3小时约40条消息
- Plus用户限制

### 注意事项

1. **信息准确性**
   - ⚠️ 可能产生"幻觉"（编造信息）
   - ✅ 关键信息需要验证
   - ✅ 引用来源时要核实

2. **隐私安全**
   - ❌ 不要输入敏感信息
   - ❌ 不要分享密码、密钥
   - ❌ 注意商业机密

3. **版权问题**
   - ⚠️ AI生成内容的版权归属不明确
   - ✅ 商业使用需谨慎
   - ✅ 参考而非抄袭

4. **道德使用**
   - ❌ 不用于作弊
   - ❌ 不生成有害内容
   - ✅ 辅助而非替代思考

## 🔧 常见问题

### Q: ChatGPT在国内怎么用？

**A**: 
1. 使用魔法上网访问官网
2. 使用国内镜像站（注意安全）
3. 使用API调用
4. 使用国产替代品（文心一言、Kimi等）

### Q: Plus值得购买吗？

**A**: 取决于需求：
- ✅ 值得：重度使用、需要GPT-4、处理图片文档
- ❌ 不急：偶尔使用、主要对话、预算有限

### Q: 如何提高回答质量？

**A**: 
1. 提示词要清晰具体
2. 提供充足上下文
3. 使用结构化提示
4. 多轮对话引导
5. 要求展示思考过程

### Q: 遇到错误回答怎么办？

**A**:
1. 直接指出错误
2. 要求重新思考
3. 提供正确信息
4. 换个提问方式

### Q: 如何保存对话？

**A**:
1. 对话会自动保存在历史记录
2. 可以分享对话（生成链接）
3. 复制粘贴到本地保存
4. 使用浏览器插件导出
